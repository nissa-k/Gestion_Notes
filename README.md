# Gestion des notes en C#

## Description

Ce projet est une application console développée en C# avec Visual Studio 2022 dans le cadre d’un travail pratique.

Le programme permet de saisir un ensemble de notes, puis d’effectuer différentes opérations d’analyse via un menu interactif : affichage, statistiques, recherche de valeurs particulières, etc.

Il a pour objectif de mettre en pratique :

- la manipulation des tableaux
- les boucles
- les conditions
- les menus en console
- les calculs statistiques simples

---

## Fonctionnalités

Après la saisie du nombre de notes et des valeurs, l’utilisateur peut accéder à un menu proposant :

1. Afficher toutes les notes  
2. Afficher le nombre de notes supérieures à 10  
3. Afficher toutes les notes inférieures à 10  
4. Afficher la moyenne des notes  
5. Afficher la meilleure note  
6. Afficher la moins bonne note  
7. Vérifier s’il existe une note égale à 20  
8. Vérifier s’il existe une note inférieure à 10  
9. Quitter le programme  

Chaque option est traitée dynamiquement à partir du tableau de notes saisi par l’utilisateur.

## Installation et exécution

1. Cloner le dépôt GitHub ou télécharger le projet
2. Ouvrir le fichier `TP6GestionNotes.sln` (ou `.csproj`) avec Visual Studio 2022
3. Compiler le projet
4. Lancer avec :

   - F5 (avec débogage)
   - ou Ctrl + F5 (sans débogage)

## Exemple d’utilisation

1. L’utilisateur saisit le nombre de notes
2. Il entre chaque note une par une
3. Le menu s’affiche
4. Il choisit une option (statistiques, affichage, recherche…)
5. Le programme affiche le résultat
6. Retour au menu jusqu’à la sortie
